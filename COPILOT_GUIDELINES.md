### Copilot Guidelines

Best practices for effectively utilizing GitHub Copilot within the project.

1. **Effective Prompting**
   - Use clear and concise instructions.
   - Provide examples to guide the AI.

2. **Review Suggestions**
   - Always review and test Copilot-generated code.

3. **Contextual Documentation**
   - Integrate comments and documentation for clarity.

4. **Feedback Loop**
   - Encourage refining prompts to improve output quality.

---

Consider combining Copilot-assistance with manual review processes to ensure high-quality contributions.
